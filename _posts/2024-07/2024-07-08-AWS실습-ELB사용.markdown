# 📌 Elastic Load Balancer
- 로드 밸런서를 사용하는 이유?<br>
    1) 요청분산(EC2와 연동)
    2) 단일 엑세스 포인트 공개(Route 53과 연동)
    3) 인스턴스에 대한 헬스체크
    4) HTTPS 제공(ACM과 연동)
    5) 고가용성 제공
    6) 공개 트래픽과 내부 트래픽 분리
